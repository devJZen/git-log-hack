# PR 날짜 조작 실험

## 실험 목적
Pull Request에 포함된 커밋의 날짜를 조작하면 GitHub 잔디밭에 어떻게 반영되는지 확인

## 실험 설계

### 테스트 1: 과거 날짜 커밋 → PR
- 브랜치: `test-pr-date`
- 커밋 날짜: 2024-02-14 (발렌타인데이)
- 실제 생성 날짜: 2024-12-31
- PR 생성 후 잔디밭 확인

### 예상 결과
- GitHub 잔디밭에 **2024-02-14**에 커밋이 표시됨
- PR 생성/머지 날짜는 잔디밭에 영향 없음
